Meet The Django Book

Welcome to the online version of The Django Book, a free book about Django.

The book is due to be published in 2007 by Apress, but in the meantime you can read the "beta" book online. This is a work in progress, and we invite you to leave comments, suggestions and criticism (read more about the comment system).

So, dive in, let us know what you think, and check back regularly for the latest chapter updates. There's an Atom feed that'll get updated each time we release new material.

Note: if you want to follow along with the examples, you'll need to install the latest development version of Django (not version 0.95, which is the latest official release).

If you've got more general feedback that doesn't fit into the comment system, or if you just like email more, you can reach us at feedback @ this domain.
